ASCII-Scramble

ASCII-Scramble is a shoot'em up for MS DOS text mode using only ASCII characters. It also works with Windows provided you know how to get an MS DOS prompt or command line. For other platforms, you can use DOSBox.

Download ASCSCRAM.ZIP

Main features are:

  • needs only 7 bit ASCII characters and 3 different colors to render epic space battles
  • no annoying sound
  • tiny EXE file (less than 40k)
  • requires about 10k of memory at run time
  • runs at usable speed even with original 4.77 MHz XTs

For installation, controls and more information, read the README.TXT.
